# Relaxed MEMs

This repo contains all code necessary to replicate the results in the Relaxed MEMs paper. 

- Functions_RelaxedMEM.R: provides all functions needed to run relaxed MEMs (binary and continuous)
- Run_Arbitraryk.R: Runs the arbitrary k simulations and exports results
- Run_MSEbased.R: Runs the MSE based simulations and exports results
- Figures_Arbitraryk.R: Creates all figures for the arbitrary k simulation results
- Figures_MSEbased.R: Creates all figures for the MSE based simulation results
- CaseStudies.R: Runs the CF case studies
